About Blood Bank Management System

The Blood Bank Management System is a web-based application designed to manage blood donors, blood stock, and blood requests efficiently. It helps in maintaining donor records and tracking available blood groups.

This system reduces manual work and ensures quick access to blood during emergencies. It provides an easy-to-use interface for users to register as donors and request blood.

The project is developed using HTML, CSS, and JavaScript, focusing on responsive design and smooth user interaction. It improves efficiency in managing blood bank operations.
